# Political Leader Codex

A static website for a political leader built with React and Vite.

## Features

- Multi-language support: English, Hindi, Marathi
- Dark and light theme toggle
- Static content from constants
- Images served from public folder

## Setup

1. Install dependencies: `npm install`
2. Start development server: `npm run dev`
3. Build for production: `npm run build`

## Customization

- Update leader data in `src/constants.js`
- Add translations in `src/i18n.js`
- Replace placeholder image in `public/images/leader.jpg`
- Modify styles in `src/App.css`
